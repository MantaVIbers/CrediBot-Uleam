# Política de crédito CrediBot MVP

Este documento es la fuente de verdad para respuestas informativas del bot.
Los datos son ficticios y se usan solo para fines académicos.

## Requisitos básicos

Para solicitar una precalificación, el usuario debe entregar nombre completo,
cédula ecuatoriana válida, destino del crédito, monto solicitado, plazo deseado
e ingreso mensual aproximado.

Antes de consultar el perfil crediticio, el usuario debe autorizar el uso de sus
datos. Sin autorización, el bot no puede continuar la precalificación.

## Documentos referenciales

La precalificación no reemplaza la revisión final de un asesor. Para continuar
con una solicitud formal, un asesor puede pedir cédula, comprobante de ingresos,
planilla de servicio básico y documentos de respaldo según el destino del crédito.

## Montos y plazos

El flujo académico permite evaluar plazos entre 3 y 36 meses. El monto máximo se
calcula con reglas internas usando score, ingreso, cuotas vigentes y capacidad de
pago.

## Tasas y cuota

La cuota estimada se calcula con amortización francesa y una tasa referencial
según la categoría del score. La tasa mostrada por CrediBot es referencial y puede
cambiar en la validación final con un asesor.

## Resultados posibles

Preaprobado significa que el perfil cumple las reglas automáticas básicas.
Observado significa que el caso necesita revisión humana.
No cumple significa que existe una condición de riesgo como score alto riesgo,
mora activa mayor a 30 días o lista negra.

## Derivación humana

El usuario puede pedir hablar con un asesor en cualquier momento escribiendo
asesor, humano, persona, agente o ejecutivo.
